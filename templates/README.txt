# Plantillas de VirtualHost

Coloca aquí las plantillas de configuración para Apache VirtualHost.

- generic.txt: Plantilla genérica para cualquier proyecto PHP.
- laravel.txt: Plantilla optimizada para proyectos Laravel (usa /public como DocumentRoot).
- wordpress.txt: Plantilla para WordPress.
- miniphp.txt: Plantilla para proyectos PHP simples.

Puedes agregar nuevas plantillas siguiendo el formato de placeholders:
- {{DOMAIN}}
- {{DOCROOT}}
- {{LOGS_DIR}}
- {{SSL_KEY}}
- {{SSL_CERT}}
